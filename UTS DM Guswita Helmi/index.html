Kisah Saya

ditulis oleh Guswita Helmi. Pada 11 April 2023

Nama saya Guswita Helmi adalah seorang mahasiswi Universitas Adzkia, jurusan Informatika semester IV. Sedikit bercerita tentang kisah nya masuk Adzkia, sebenarnya ambil jurusan Informatika bukan karena punya kemampuan disitu, tetapi pada waktu itu hanya itu kata-kata yang menarik, INFORMATIKA. Modal nekad saja sebenarnya.

Sedikit cerita, saya bercita-cita menjadi seorang TNI. Saya sama sekali tidak berniat untuk berkuliah karena saya mau fokus untuk tes saja. Tahun 2020, setelah lulus SMA saya memutuskan untuk mengikuti pembinaan di Koramil. Pada saat latihan saya sangat fokus dan bersungguh-sungguh agar hasilnya maksimal. Singkat cerita saya mengikuti tes Bintara TNI AL atau Kowal untuk wanita. Saya sangat percaya diri mengikuti tes demi tes. Pada suatu hari pengumuman hasil parade tingkat daerah saya dinyatakan gagal. Disitu saya merasa sangat sedih dan kecewa karena kegagalan yang saya terima. Setelah beberapa bulan saya kembali bangkit untuk tes kembali, namun ternyata kedua orang tua saya tidak mengizinkan saya untuk tes kembali.

Setelah tahu bahwa saya tidak diizinkan tes kembali, saya berusaha mencari alternatif supaya tetap bisa ikut tes kembali. Suatu hari guru SMA saya menawarkan kuliah di Universitas Adzkia di padang. Saya sangat gembira untuk itu karena Allah beri saya jalan. Saya ikut seleksi Beasiswa Unggul Adzkia dan lolos. Namun takdir berkata lain, pada saat tanda tangan kontrak saya sangat terkejut karena salah satu syaratnya adalah Ijazah SMA asli ditahan. Saya sanagt tidak terima, dalam hati saya berkata mundur saja tidak perlu ambil beasiswa nya masih banyak jalan lain. Namun, saya kembali tersadar pada saat itu saya membawa ibu dan adik saya kesana. Mereka sangat bahagia karena mengetahui saya akan kuliah. Saya mengurungkan niat dan ego saya untuk mundur. Saya kubur dalam cita-cita saya demi kebahagiaan mereka yang saya sayangi.

Sekarang saya sudah mulai menikmati masa-masa menjadi mahasiswa karena saya mendapat banyak hal baru. Saya memiliki kebahagiaan baru disini, ternyata kuliah tidak semembosankan itu. Namun demikian saya tetap akan mengikuti tes TNI nanti setelah mendapat gelar sarjana. Aamiin.
